# Reporting Issues

PromptX takes reliability, security, and responsible use seriously.

If you encounter any of the following:
- Suspected security vulnerabilities
- Potential misuse of APIs
- Unexpected or unsafe AI behaviour
- Data handling or access concerns
- Platform errors impacting integrations

Please report the issue promptly so it can be reviewed and addressed.

## What Happens Next?

Once an issue is reported:
- It will be triaged based on severity and impact
- Relevant teams will investigate and respond as appropriate
- You may be contacted for additional information

Reporting issues responsibly helps keep PromptX safe and reliable for everyone.
