API Access Models

PromptX supports two primary API access models.

Human User Tokens

Used when individuals invoke APIs for advanced workflows or tooling.

Service Accounts

Used when systems or applications invoke APIs programmatically.

Each model has different: - Approval requirements - Monitoring expectations - Risk profiles
