API Reference

PromptX API reference documentation provides detailed technical specifications for approved integrations, including endpoint definitions, request and response schemas, and usage constraints.

This developer portal intentionally focuses on principles, governance, and safe usage patterns.
Detailed endpoint-level documentation is shared only with approved consumers to ensure alignment with organisational policy and risk controls.

Request Detailed API Documentation

If you require access to detailed API specifications, including endpoint definitions and schemas, please submit a documentation request.

How Documentation Access Works

Once your request is received: - The request will be reviewed for relevance and risk - Documentation access may be scoped to specific capabilities - Updated versions may be shared as APIs evolve

Documentation access does not imply unrestricted API access and may be time-bound.
